Widget to place today's date on your website in a convenient location.

== Description ==

Using a widget you're able to place today's date available on your site.

== Installation ==

1. Upload `/todays-date-widget/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
